# Employee Management System - JDBC Layered Architecture

## Overview
This project is a Java JDBC application developed using Layered Architecture. It demonstrates CRUD operations on employee records stored in a PostgreSQL database.

## Features
- Save Employee
- Update Employee
- Delete Employee
- Fetch Employee by ID
- Fetch All Employees

## Technologies Used
- Java
- JDBC
- PostgreSQL
- Maven
- Eclipse IDE

## Project Structure
- EmployeeController.java
- EmployeeDao.java
- EmployeeEntity.java
- DBConnection.java

## Database
PostgreSQL
